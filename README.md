# Simple To-Do List

This is a simple and beginner-friendly To-Do List web app built using **React.js**. It allows users to add, complete, undo, and delete tasks.

---

## Features

- Add new tasks
- Mark tasks as complete or undo them
- Delete tasks
- Data saved with `localStorage`

---

## Technologies Used

- React.js (Vite)
- JavaScript
- HTML/CSS
- LocalStorage
